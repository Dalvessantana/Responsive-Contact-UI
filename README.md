# 📱 Responsive Contact UI

Uma interface de formulário de contato moderna, elegante e **totalmente responsiva**. Este projeto foi desenvolvido com a metodologia de adaptação de layout, garantindo que a experiência do usuário seja consistente tanto em monitores de alta resolução quanto em dispositivos móveis.

## 🎯 Destaque: Responsividade
Diferente de layouts estáticos, este projeto utiliza técnicas avançadas de CSS para reorganizar o conteúdo conforme o tamanho da tela:
- **Desktop:** Layout em duas colunas (Informações à esquerda, formulário à direita).
- **Mobile/Tablet:** Transição suave para layout vertical, otimizando o espaço e facilitando o toque nos campos.

## ✨ Diferenciais do Projeto

- **Flexbox & Grid:** Uso de modelos de layout flexíveis para evitar quebras de design.
- **Formulário Otimizado:** Inputs com tamanhos adaptáveis e botões fáceis de interagir em telas touch.
- **UI Elegante:** Paleta de cores Indigo/Violeta com alto contraste e legibilidade.
